Documentation technique PAD+

La PAD+ est une extension de Moodle qui conjugue trois piliers :

  • un modèle d'organisation du contenu et des rôles associés ;
  • une configuration spécifique d'une trentaine de paramètres ainsi que des permissions ;
  • des extensions (plugins) et quelques modifications du code Moodle open source.

Ce site présente les éléments pour l'installation, l'organisation et le développement d'une instance PAD+. Elle complète la documentation Moodle avec les spécificités de la PAD+ et fournit des pointures utiles vers celle-ci. Pour les aspects génériques de Moodle, se référer à sa documentation propre.
